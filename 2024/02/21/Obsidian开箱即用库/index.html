Obsidian开箱即用库

发表于 2024-02-21

前言

Obsidian 相信大家应该不陌生了，一款使用 Markdown 语法的双链笔记工具。个人免费使用，丰富的插件系统，知识快速链接，都是其出色的地方。当然它也有一些值得诟病的地方，例如界面简单，新手打开感觉就是一个普通笔记本，和隔壁 Notion 的初始的精美相比，实在是“简陋”。因此大部分小伙伴都是了解一段时间后要么放弃，要么转战其他笔记工具。因此这篇文章就是告诉大家如何利用丰富的插件库来打造专属个人的笔记本。文章最后会附上懒人包，开箱即用。

准备

可以到官网下载对应平台的安装包，几乎做到了全平台覆盖，附上官网地址。

建立工作区

  1. Create new vault ：新建一个工作区，如果之前没有使用过就新建一个工作区，用于存放对应的 markdown 文件。
  2. Open folder as vault ：如果之前有建立过工作区，或者从其他人那里获取到了模板，可以通过这个选项打开。
  3. Open vault from Obsidian Sync ：如果之前升级过个人账户，可以通过这个选项同步远端笔记（免费版也可以通过插件做到）。
    语言选择可以自己切换，支持简体中文。

新建笔记

点击 文件列表 中的 新建笔记 即可创建一篇新的笔记，也可以通过 cmd + p的快捷键打开控制台搜索 新建笔记 的命令来创建。文件创建后为 .md 格式文件，因此支持 md 语法，不清楚该语法的朋友可以参考这里。

不同的笔记之间支持双链，双链笔记的好处不在这篇文章的探讨范围，有兴趣可以参考

插件生态

基础的笔记功能就是这样，如果只看到这里，可能觉得这个工具也并没有什么出彩的地方。接下来就进入更容易引起大家兴趣的部分，如何使用其丰富的插件库来打造个人的工作区。

笔记核心

笔记到底记什么，我想每个人都有自己不同的看法，找到属于自己的记笔记的方式需要一段时间的探索。那么作为刚入门的萌新，我们可以参考其他人记笔记的方式来学习，模仿也是一种学习。

如果你觉得别人讲的道理有理有据，而自己暂时无法反驳，碰巧自己又非常想做这件事，那就相信他们说的是对的，然后笃定地行动

我个人喜欢把笔记分为以下几个部分：

  1. 每日日记
  2. 闪念笔记
  3. 读书笔记
  4. 任务清单
  5. 文章剪藏
    接下来我会展开讲讲每部分的功能，以及如何通过插件整合到 Obsidian 中。

日记

日记大家肯定都不陌生，小时候可能就被老师要求过每日写日记，但是却没想过写日记对我们来说有什么好处。

  1. 增强注意力。现在每天都有很多琐碎的事情，写日记的时候需要你专注回想一天经历的有意思的事情，提升专注能力
  2. 实现目标。很多人在年初的时候制订了很多目标，但是无法跟踪，只能在年底看结果，而日记可以通过分解并跟踪目标，从而更清晰目标的完成情况
  3. 记忆力和理解能力。好记性不如烂笔头，将事情记录下来相当于在脑袋里面又加深了一次印象
  4. 缓解压力。用语言文字“给情绪贴标签”，能启动大脑情绪自我控制功能，进而缓解紧张，释放压力

日记，顾名思义，记录当天发生的有意思的事情。这是我的每日日记模板，参考了B站一位朋友的模板，做了一些自己的调整，文章最后会把他创建的懒人包也分享出来。

其中分为 今日主题， 感恩三件事， 任务清单， 随笔感悟， 读书笔记

今日主题

今天发生的三件影响自己的事情，例如和许久未见的好朋友见面，认识了一位很感兴趣的新朋友，或者完成了一件很重要的事情。只要是影响到自己的事情都可以记录，如果没有发生，也不强求填写，写日记需要放松，而不是做科研。
最后可以总结一下今天的感受，几句话就行，不需要正式的语气，按照个人习惯就好。

感恩三件事

我们知道世界各地都有和感恩相关的节日，例如美国的“感恩节“，韩国的”仲秋节“，以色列的”住棚节“。中国也有”父亲节“，”母亲节“，”教师节“，并且自古以来就教育人们要学会感恩，”滴水之恩，涌泉相报“。那么人为什么要感恩？表达感恩，是积极心理学的一部分。感恩心态能让人更积极、正面，凡事往好处想，而不是总被一些烦躁的事情困住，从而影响睡眠。
这里的感恩三件事就是要记录下让你有感恩想法的任何其他事情，可以是人（xx帮助我做了一件xx事），可以是物（xx工具帮助了我），可以是风景（xx地方的风景很漂亮，让我心情很好），也可以是天气（xx天气是我最爱的）。

任务清单

这里使用的是 Obsidian 社区插件中的 Tasks，可以设置对应的创建时间，优先级，开始时间，结束时间等。可以通过其提供的语法很快速的查询对应的任务。

待完成
通过插件提供的语法，查询之前遗留的任务。如果任务不重要，可以重新设置完成时间或者直接关闭任务。

Today
今天创建的任务，通过加入 QuickAdd 插件，可以实现一个快捷键将任务插入到当前页面。

随笔感悟

不知道大家有没有这样的经历，当你专注在做一件事情的时候，脑袋里蹦出了另一个想法，于是你立马转手去做了另一件事情，导致本来专注的状态被打破。或者突然间有了一个很好的想法，过了一段时间却怎么也想不起来了。这种情况就需要随手记录下来。写下来以后你就不会那么着急去马上完成，因为你知道可以完成手中的事情后再去回顾它。许多散碎的想法拼接在一起可能就能形成新的灵感，这就是闪念笔记的好处。
这里我使用的是插件 Memo，设置对应的快捷键，快速的打开并且记录到 Memo 窗口，并且会自动同步到每日笔记的 随笔感悟 中，这样每天查看日记的时候，就能捡回那些灵感，或者回忆起某件事情。

读书笔记

Obsidian 有微信读书的插件，会把你对应书的高亮和评论同步过来，并且通过设置，可以同步当天的高亮和评论到你的每日日记中，这样每天阅读日记的时候也可以回顾一下这些影响你的句子或者想法。

除了每日日记，也可以设置周度日记，月度日记，年度日记。

这是我的周度笔记，通过插件 Templater + Dataview 可以实现自动从这一周的日记中抓取信息。在每日笔记中添加 🍅 tag，可以记录每周番茄工作法的时间。

文章剪藏

Obsidian 的 Ominivore 插件可以实现同步文章，通过Dataview插件就可以实现每周笔记中看到的剪藏文章汇总的模块。

其他

ChatGPT

通过插件 ChatGPT MD 实现了 OpenAI API 查询。需要输入用户自己的密钥，可以搜索如何获取 OpenAI 密钥。

Periodic Notes

该插件可以实现日度、周度、月度、季度、年度笔记。已经整合到懒人包，通过命令打开对应的笔记。

Spaced Repetition

该插件可以实现快速回顾，帮助你快速回顾。只需要在笔记中加入 ?, :: 等，即可自动生成卡片，每天可以回顾卡片。

Excaildraw

画图工具，个人感觉比白板好用，并且网上有很多素材可以使用。也是通过命令即可新建图。
